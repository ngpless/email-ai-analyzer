попадает в **пересечение двух категорий** Приложения 8:

Нужно соответствовать обоим одновременно.

## Про использование ИИ (критично!)

Если при написании ВКР использовались GPT-подобные системы, то по
методичке:

1. **Сгенерированный текст/код выделить синим цветом и курсивом** в ВКР.
2. Во Введении указать название модели СГИИ, использованный промпт,
   обосновать цель и способы применения.
3. В списке литературы **первым источником** — ссылка на модель.
4. Источники фактов подтвердить другими, не СГИИ-источниками. Ссылки
   в квадратных скобках `[2, стр. 5]`.

Только при соблюдении всех четырёх пунктов руководитель в отзыве
ставит отметку об использовании СГИИ.
