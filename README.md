## Тема

Разработка AI-приложения для анализа почтовых сообщений.

## Цель работы

Спроектировать и реализовать прототип клиент-серверного приложения, которое
автоматически анализирует почтовые сообщения пользователя и выдаёт:

- классификацию по категориям (работа / личное / реклама / важное …)
- обнаружение спама и фишинга
- суммаризацию длинных писем
- извлечение сущностей (даты, суммы, контакты, задачи)
- семантический поиск по почтовому архиву

## Архитектура

Клиент-серверная:

- **Backend** — FastAPI + SQLAlchemy + PostgreSQL (dev: SQLite)
- **ML-ядро** — scikit-learn (TF-IDF + LogReg) с интерфейсом для замены
  на BERT/LLM-API без изменения кода потребителей
- **Mail** — IMAP-клиент на стандартной библиотеке `imaplib`
- **Client** — десктоп на PyQt6
- **Тесты** — pytest

Подробнее: [`docs/architecture.md`](docs/architecture.md)

## Запуск

```bash
# Установить зависимости
pip install -r requirements.txt

# Инициализировать БД (создаст SQLite-файл и тестовых пользователей)
python scripts/init_db.py

# Запустить backend
uvicorn email_analyzer.backend.main:app --reload

# В другом терминале — клиент
python -m email_analyzer.client.main

# Запустить все тесты
pytest -v

# Собрать .exe
python scripts/build_exe.py
```
